【解决方案】Visual Studio写C/C++时没有语法高亮

发表于2023-04-05|更新于2023-04-05|技术
|字数总计:66|阅读时长:1分钟|阅读量:

最近突然用VS写了一下C，发现没有语法高亮了，非常难受，如图

解决方案如下
在 工具->选项->文本编辑器->C/C++->高级中
将禁用语义着色设为False
然后就恢复了
